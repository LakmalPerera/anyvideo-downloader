# Universal Video Downloader

A modern web-based video downloader built with Node.js and yt-dlp.

## Features
- Download videos from multiple platforms
- Quality selection (360p / 480p / 720p / 1080p)
- MP3 audio-only downloads
- Real-time progress updates
- Modern dark UI
- Toast notifications on completion

## Tech Stack
- Node.js
- Express
- yt-dlp
- HTML / CSS / JavaScript

## How to Run Locally
```bash
npm install
node server.js
